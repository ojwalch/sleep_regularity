Calculate sleep regularity index from actigraphy files

This tool is currently under construction. It assumes inputs of .csv files exported from Actiware and an epoch size of 30 seconds.

About

This website reads actigraphy data files (in csv format) to calculate the sleep regularity index for each participant. You can download the template for your actigraphy data here and sample data here. We don't store any of the data you upload.
